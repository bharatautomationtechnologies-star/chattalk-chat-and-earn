# ChatTalk - Talk to Strangers & Earn Coins

ChatTalk is a social chat application where you can meet new people from around the world, enjoy interesting conversations, and earn virtual coins by participating in chat sessions.

## Features

- 💬 Talk to Strangers Worldwide
- 🌍 Meet New Friends
- 🪙 Earn Virtual Coins Through Chat
- 🎁 Redeem Coins for Available Rewards
- 👥 Refer Friends & Earn More Coins
- 💼 Wallet & Coin History
- 🌙 Dark Mode
- 🔒 Secure Login
- 📱 Simple & User-Friendly Interface
